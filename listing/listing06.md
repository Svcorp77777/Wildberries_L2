Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
[3 5 3]

В main будет создан слайс s с элементами {"1", "2", "3"}.
Далее слайс s передается в функцию modifySlice

При передаче слайса в функцию, сам слайс передается по значению, но он по-прежнему
ссылается на тот же массив данных. Поэтому изменения в элементах слайса видны
как внутри функции, так и снаружи. Но если внутри функции произодет изменение длины
слайса например при использовании append, то будет создан новый слайс который уже не
будет иметь отношение к изначальному слайсу.
Так происходит и в данном кейсе, мы передаем слайс в функцию modifySlice и происходит следующее:
i[0] = "3" - тут изменяется наш исходный слайс (то есть, он уже выглядит так "3", "2", "3")
Затем вызывается i = append(i, "4"), это создает новый слайс
i[1] = "5" - уже никак не влияет на наш старый слайс, оно изменяет только новый созданный

Слайс в Go представляет собой динамический массив с информацией о длине и емкости. 
Внутренне слайс состоит из трех компонентов:
1. Указатель на массив данных
2. Длина слайса
3. Емкость слайса
```
