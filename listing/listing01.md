Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]

Будет создан массива а размер которого пять элементов целочисленного типа:
{76, 77, 78, 79, 80}.
Далее будет создан срез b целочисленного типа, который включает элементы
массива а с индекса 1 включительно, по 4 индекс не включая.
Соответственно срез b будет содержать такие элементы: {77, 78, 79}.
```
